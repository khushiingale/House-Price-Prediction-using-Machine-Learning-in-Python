# 🏠 House Price Prediction using Machine Learning

## 🚀 Overview
This project aims to predict house prices using machine learning techniques based on various features such as location, size, condition, and other property-related factors. It helps buyers and sellers make informed decisions in the real estate market.

---

## 🎯 Features
- 📊 Data preprocessing and cleaning  
- 📈 Exploratory Data Analysis (EDA)  
- 🔥 Correlation analysis using heatmaps  
- 🏡 Feature-based price prediction  
- 🤖 Multiple regression models comparison  
- 📉 Model evaluation using error metrics  

---

## 🛠️ Tech Stack
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn

## ⚙️ How It Works
1. Load dataset with housing features  
2. Clean data (handle missing values, drop irrelevant columns)  
3. Perform exploratory data analysis  
4. Convert categorical data using encoding  
5. Split dataset into training and testing sets  
6. Train machine learning models  
7. Evaluate model performance
